# مشروع مركز الابتكار - هيئة الحكومة الرقمية (DGA Innovation Hub)

هذا المستند مخصص لنقل سياق المشروع إلى مساعد الذكاء الاصطناعي (Antigravity) في بيئة عمل جديدة.

## 📌 ملخص المشروع
المشروع عبارة عن واجهة رقمية تفاعلية فاخرة (Premium Luxury Editorial) لمركز الابتكار، تعتمد على التمرير (Scrolling) لاستعراض مختبرات المركز عبر مقاطع فيديو سينمائية مع واجهات زجاجية متطورة.

## 🛠 التقنيات المستخدمة
- **Core**: HTML5, CSS3, Vanilla JavaScript.
- **Graphics**: WebGL Shaders (للخلفيات المتحركة).
- **Styling**: Glassmorphism, Advanced CSS Filters, Responsive Design.
- **Deployment**: Vercel.
- **Source Control**: GitHub.

## 📂 هيكل المجلدات الرئيسي
- `index.html`: الملف الرئيسي الذي يحتوي على كامل هيكل الصفحة، التصميم (CSS)، والمنطق (JS).
- `اللوقوهات والخط والالوان/`: يحتوي على أصول الهوية البصرية (الخطوط واللوقوهات).
- المجلدات المرقمة (01 إلى 06): تحتوي على مقاطع الفيديو السينمائية لكل مختبر.
- `src/index.css`: ملف CSS إضافي (اختياري).

## ✨ المميزات الحالية (Current State)
1. **نظام التمرير السينمائي**: فيديوهات HLS/MP4 تعمل بسلاسة مع التمرير وتتحكم في توقيت المشاهد.
2. **واجهة Animated Glassy**: صناديق وصف المختبرات تعتمد تصميم "Liquid Glass" مع شفافية عالية (Blur 14px) وتأثيرات بصرية متميزة.
3. **خلفية الشعاع الضوئي (WebGL)**: شعاع متحرك من اليمين لليسار بألوان الهيئة (بنفسجي وسماوي) يظهر خلف صناديق المختبرات.
4. **البداية والنهاية**:
   - **Intro**: لوقو الهيئة الملون مع أنيميشن زووم سينمائي (Journey to the future) بدون إضاءة مشتتة.
   - **Outro**: صفحة شكر بسيطة وأنيقة تحتوي على باركود (QR Code) وزر لحجز المختبرات.
5. **المؤشر**: تم العودة للماوس الأصلي للنظام مع إضافة هالة ضوئية بيضاء/سماوية صغيرة جداً تتبعه بسلاسة.
